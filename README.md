# Periscope

A Docker and Kubernetes management tool designed to simplify container orchestration and cluster management.

## Features

- **Docker Management** - Manage containers, images, and volumes with ease
- **Kubernetes Integration** - Seamless integration with Kubernetes clusters
- **Simplified UI** - Intuitive interface for managing complex containerized environments
- **Real-time Monitoring** - Monitor containers and pod status in real-time
